api.othub.io Documentation
The OTHub beta API is a direct line to the Origintrail Decentalized Knowledge Graph Network as well as the Origintrail Parachain ecosystem.

The API offers:

Complete dkg.js SDK free access via OTHub Origintrail nodes.
Access to DKG data as well as parachain and gnosis asset data (inventories, history)
OTHub's create-n-transfer call to mint and transfer assets with no crypto or wallet needed
Certain features of the API require an end user to sign for transactions via a web3 wallet integration (MetaMask) at https://www.othub.io/my-othub/portal ,where your users will see specific transactions the API key owner has queued for them.

dkg
This contains all API requests that require a dkg node. All requests are routed through OTHub's DKG node. We do not offer the ability to create/query/get private data.

All publish,update, and transfer transactions are queued on OTHub's portal for the end user to sign and approve in metamask.
